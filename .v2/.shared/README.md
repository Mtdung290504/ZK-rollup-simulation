In practice, components are fully separated, from source code to programming languages. However, in the simulation environment, shared modules in this directory are reused to improve convenience and reduce code duplication.

Trong thực tế, các thành phần được tách biệt hoàn toàn, từ mã nguồn đến ngôn ngữ lập trình. Tuy nhiên, trong môi trường mô phỏng, các module chung trong thư mục này được tái sử dụng nhằm tăng tính tiện lợi và giảm trùng lặp mã.
